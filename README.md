##  Installation 

Setup .env variables
    - database name anime_db
    - your database user name
    - your database password

Create your database

seed your database
    ```
    npm run seed
    ```

Start server
    ```
    npm start
    ```

View locally
    ```
    localhost:3001
    ```

Or visit the deployed application
